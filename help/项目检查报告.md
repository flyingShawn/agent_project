## 🚀 快速开始指南

### 方式一：本地测试（推荐新手）

#### 第一步：配置环境

1. 复制配置文件：
   ```bash
   copy .env.example .env
   ```

2. 编辑配置文件，填写数据库连接信息

#### 第二步：运行快速测试脚本

双击运行 `快速测试.bat`，脚本会自动检查：
- Python环境
- 依赖安装
- 环境配置
- 后端启动测试

#### 第三步：启动服务

1. **启动Ollama**（新开一个终端）：
   ```bash
   ollama serve
   ```

2. **下载模型**（首次需要）：
   ```bash
   ollama pull qwen2.5:7b
   ollama pull qwen2.5-vl:7b
   ```

3. **启动后端**：
   ```bash
   python -m uvicorn agent_backend.main:app --reload
   ```

4. **启动前端**（新开一个终端）：
   ```bash
   cd agent_frontend
   npm install
   npm run dev
   ```

5. **访问服务**：
   - 前端界面：http://localhost:3000
   - API文档：http://localhost:8000/docs

---

### 方式二：Docker部署

#### 一键部署

双击运行 `docker\deploy.bat`

#### 手动部署

```bash
# 构建镜像
docker-compose build

# 启动服务
docker-compose up -d

# 下载模型（首次需要）
docker exec -it desk-agent-ollama ollama pull qwen2.5:7b
docker exec -it desk-agent-ollama ollama pull qwen2.5-vl:7b
```

#### 访问服务

- 前端界面：http://localhost
- API文档：http://localhost:8000/docs
- Qdrant控制台：http://localhost:6333/dashboard

## ⚠️ 注意事项

### 1. 模型下载

**重要**：首次运行需要下载大模型，文件较大（约4-7GB），请耐心等待。

**推荐模型**：
- 文本模型：`qwen2.5:7b` (约4.7GB)
- 视觉模型：`qwen2.5-vl:7b` (约4.7GB)

**如果内存不足**，可以使用更小的模型：
- `qwen2.5:3b` (约2GB)
- `qwen2.5-vl:3b` (约2GB)

### 2. 数据库连接

**当前状态**：需要在 `.env` 文件中配置数据库连接

**配置示例**：
```bash
DB_TYPE=mysql
DB_HOST=localhost
DB_PORT=3306
DB_NAME=your_database
DB_USER=root
DB_PASSWORD=your_password
```

### 3. 文档知识库

**当前状态**：`data/docs/` 目录下只有一个示例文档

**添加文档**：
1. 将PDF、Word、TXT等文档放入 `data/docs/` 目录
2. 调用API同步文档：
   ```bash
   curl -X POST http://localhost:8000/api/v1/rag/sync
   ```
