你是一位专业的ai agent开发专家，精通解决智能体开发问题。

这是为老C++桌面管理系统开发的智能Agent助手，提供自然语言"知识文档问答"及"数据库只读数据智能查询"功能，现在需要你帮我解决一些存在的问题。

此工程采用技术栈

| **层级**       | **技术**                                |
| -------------------- | --------------------------------------------- |
| **Web框架**    | **FastAPI + Uvicorn**                   |
| **数据验证**   | **Pydantic 2.0+**                       |
| **数据库**     | **SQLAlchemy 2.0**                      |
| **向量数据库** | **Qdrant**                              |
| **向量模型**   | **BAAI/bge-m3 / bge-small-zh-v1.5**     |
| **大语言模型** | **Ollama + Qwen2.5:7b / Qwen2.5-vl:7b** |
| **工作流编排** | **LangGraph**                           |
| **文档解析**   | **Docling 2.0+**                        |
| **前端框架**   | **Vue 3 + Vite + Tailwind CSS**         |
| **容器化**     | **Docker + Docker Compose**             |

### 架构模式

* **整体架构**：前后端分离的单体应用（适合当前规模，易于部署和维护）
* **后端架构**：分层架构（API层 → 业务逻辑层 → 数据访问层）
* **设计风格**：模块化设计，各功能模块相对独立
