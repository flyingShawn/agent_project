## 核心模块

### 1. 意图路由 (`chat/router.py`)

**基于关键词 + 正则的评分机制，将用户问题路由到 SQL 或 RAG 处理通道：**

* **SQL 意图**：查询、统计、数量类问题（如"有多少台设备在线"）
* **RAG 意图**：操作指南、配置方法类问题（如"如何设置设备权限"）

### 2. SQL Agent (`sql_agent/`)

**自然语言转 SQL 的安全查询引擎：**

* **RAG 检索相似 SQL 样本辅助生成**
* **Schema 元数据驱动的 Prompt 构建**
* **SQL 安全校验（禁止写操作、敏感列过滤、受限表管控）**
* **查询结果自然语言总结**

### 3. RAG Engine (`rag_engine/`)

**文档知识库的检索增强问答：**

* **文档入库**：Docling 解析 → Markdown 分块 → 向量化 → Qdrant 存储
* **混合检索**：向量检索 + BM25 关键词检索加权融合（alpha 可配）
* **增量更新**：基于文件指纹的增量入库，避免重复处理
* **多模态**：支持图片理解（视觉模型）

### 4. LLM 客户端 (`llm/clients.py`)

**统一封装 Ollama 调用，支持流式/同步两种模式，自动根据是否含图片切换文本/视觉模型。**

## API 路由

| **路径**             | **功能**                         |
| -------------------------- | -------------------------------------- |
| `POST /api/v1/chat`      | **统一聊天入口（自动路由意图）** |
| `POST /api/v1/rag`       | **RAG 文档问答**                 |
| `POST /api/v1/sql-agent` | **SQL 查询代理**                 |
| `GET  /api/v1/health`    | **健康检查**                     |
| `GET  /api/v1/metadata`  | **元数据查询**                   |
